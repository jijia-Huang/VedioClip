## MODIFIED Requirements
### Requirement: 播放控制
系統 SHALL 提供播放控制功能，包括播放/暫停、進度條、時間顯示等。

#### Scenario: 循環播放功能
- **WHEN** 使用者點擊「循環播放」按鈕
- **THEN** 切換循環播放狀態
- **AND** 按鈕視覺狀態更新（啟用時高亮，未啟用時一般樣式）

#### Scenario: 一般模式循環播放
- **WHEN** 循環播放已啟用且影片播放到結束
- **THEN** 影片自動跳轉到 0 秒
- **AND** 影片自動開始播放

#### Scenario: 預覽模式循環播放
- **WHEN** 循環播放已啟用且預覽片段播放到結束時間
- **THEN** 影片自動跳轉到片段開始時間
- **AND** 影片自動開始播放

#### Scenario: 關閉循環播放
- **WHEN** 循環播放已關閉且影片播放到結束
- **THEN** 影片正常停止（不自動跳轉和播放）
