## MODIFIED Requirements

### Requirement: 整體 UI 佈局
系統 SHALL 採用左右分欄佈局，左側為預覽區域，右側為片段管理和輸出設定。

#### Scenario: 左右分欄佈局
- **WHEN** 應用程式啟動
- **THEN** 顯示左右分欄佈局
- **AND** 左側區域佔據 65-70% 寬度
- **AND** 右側區域佔據 30-35% 寬度
- **AND** 頂部顯示標題列
- **AND** 底部顯示狀態列

### Requirement: 頂部標題列
系統 SHALL 在頂部顯示標題列，包含 Logo/標題、載入影片按鈕、設定按鈕、幫助按鈕。

#### Scenario: 標題列顯示
- **WHEN** 應用程式啟動
- **THEN** 頂部顯示標題列
- **AND** 左側顯示 Logo/標題
- **AND** 右側顯示載入影片按鈕、設定按鈕、幫助按鈕
- **AND** 標題列高度固定

### Requirement: 左側預覽區域
系統 SHALL 在左側顯示影片預覽視窗和播放控制。

#### Scenario: 預覽區域佈局
- **WHEN** 影片已載入
- **THEN** 左側顯示影片預覽視窗
- **AND** 預覽視窗高度最大化（佔據可用空間）
- **AND** 下方顯示播放控制（播放/暫停、進度條、入點/出點按鈕）
- **AND** 顯示當前時間和總時長

#### Scenario: 簡易播放進度條
- **WHEN** 影片正在播放或已載入
- **THEN** 顯示簡易播放進度條（Scrubber）
- **AND** 進度條可以拖動跳轉
- **AND** 顯示當前時間 / 總時長

#### Scenario: 入點/出點按鈕
- **WHEN** 影片已載入
- **THEN** 顯示入點按鈕（I 按鈕）和出點按鈕（O 按鈕）
- **AND** 點擊入點按鈕時，記錄當前播放時間為入點
- **AND** 點擊出點按鈕時，記錄當前播放時間為出點
- **AND** 入點必須小於出點

### Requirement: 右側片段管理區域
系統 SHALL 在右側顯示片段列表（表格形式）、加入片段按鈕、輸出設定。

#### Scenario: 片段列表表格
- **WHEN** 存在一個或多個片段
- **THEN** 顯示表格形式的片段列表
- **AND** 表格包含列：編號、名稱、開始時間、結束時間、操作
- **AND** 每個片段顯示編輯和刪除按鈕
- **AND** 表格可以隨視窗高度延伸（可滾動）

#### Scenario: 加入目前片段按鈕
- **WHEN** 影片已載入且設定了入點和出點
- **THEN** 顯示「加入目前片段」按鈕
- **AND** 按鈕設計醒目（大按鈕、明顯顏色）
- **AND** 點擊按鈕時，使用入點和出點建立新片段
- **AND** 如果未設定入點/出點，使用播放器當前時間作為預設值

#### Scenario: 輸出設定區域
- **WHEN** 存在片段
- **THEN** 顯示輸出設定區域
- **AND** 包含格式選擇、品質選擇、輸出目錄選擇
- **AND** 顯示批次輸出按鈕

### Requirement: 底部狀態列
系統 SHALL 在底部顯示狀態列，顯示當前應用程式狀態。

#### Scenario: 狀態列顯示
- **WHEN** 應用程式運行
- **THEN** 底部顯示狀態列
- **AND** 顯示當前狀態（就緒、載入中、匯出中等）
- **AND** 狀態列高度固定

### Requirement: 入點/出點功能
系統 SHALL 提供入點/出點功能，讓使用者快速標記時間點。

#### Scenario: 設定入點
- **WHEN** 使用者點擊入點按鈕（I 按鈕）
- **THEN** 記錄當前播放時間為入點
- **AND** 入點按鈕顯示為已啟用狀態（可選視覺反饋）
- **AND** 如果已設定出點且入點 >= 出點，顯示警告

#### Scenario: 設定出點
- **WHEN** 使用者點擊出點按鈕（O 按鈕）
- **THEN** 記錄當前播放時間為出點
- **AND** 出點按鈕顯示為已啟用狀態（可選視覺反饋）
- **AND** 如果已設定入點且出點 <= 入點，顯示警告

#### Scenario: 使用入點/出點建立片段
- **WHEN** 使用者點擊「加入目前片段」按鈕
- **THEN** 如果已設定入點和出點，使用這些時間建立片段
- **AND** 如果未設定入點/出點，使用播放器當前時間作為開始時間，結束時間為當前時間 + 預設時長（可選）
- **AND** 顯示片段編輯對話框讓使用者輸入名稱
- **AND** 建立片段後清除入點/出點（可選）

### Requirement: 響應式設計
系統 SHALL 在不同視窗大小下都能良好顯示。

#### Scenario: 最小視窗大小
- **WHEN** 視窗大小調整
- **THEN** 確保左右分欄比例維持在合理範圍
- **AND** 設定最小視窗寬度，避免佈局過於擁擠
- **AND** 表格在空間不足時可以水平滾動（可選）
