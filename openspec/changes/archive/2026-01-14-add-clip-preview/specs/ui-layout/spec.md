## MODIFIED Requirements
### Requirement: 右側片段管理區域
系統 SHALL 在右側顯示片段列表（表格形式）、新增片段按鈕、輸出設定。

#### Scenario: 片段列表表格
- **WHEN** 存在一個或多個片段
- **THEN** 顯示表格形式的片段列表
- **AND** 表格包含列：編號、名稱、開始時間、結束時間、操作
- **AND** 每個片段顯示「預覽」、編輯和刪除按鈕
- **AND** 表格可以隨視窗高度延伸（可滾動）

#### Scenario: 新增片段按鈕
- **WHEN** 影片已載入
- **THEN** 顯示「新增片段」按鈕
- **AND** 點擊按鈕時，彈出片段編輯對話框
- **AND** 使用者可以輸入名稱、開始時間、結束時間後建立片段

#### Scenario: 輸出設定區域
- **WHEN** 存在片段
- **THEN** 顯示輸出設定區域
- **AND** 包含格式選擇、品質選擇、輸出目錄選擇
- **AND** 顯示批次輸出按鈕
