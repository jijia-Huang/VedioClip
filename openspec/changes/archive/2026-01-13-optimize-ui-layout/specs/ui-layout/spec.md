## ADDED Requirements

### Requirement: 整體 UI 佈局
系統 SHALL 採用左右分欄佈局，左側為預覽區域，右側為片段管理和輸出設定。

#### Scenario: 左右分欄佈局
- **WHEN** 應用程式啟動
- **THEN** 顯示左右分欄佈局
- **AND** 左側區域佔據 65-70% 寬度
- **AND** 右側區域佔據 30-35% 寬度
- **AND** 頂部顯示標題列
- **AND** 底部顯示狀態列

### Requirement: 頂部標題列
系統 SHALL 在頂部顯示標題列，包含 Logo/標題、載入影片按鈕、設定按鈕、幫助按鈕。

#### Scenario: 標題列顯示
- **WHEN** 應用程式啟動
- **THEN** 頂部顯示標題列
- **AND** 左側顯示 Logo/標題
- **AND** 右側顯示載入影片按鈕、設定按鈕、幫助按鈕
- **AND** 標題列高度固定

### Requirement: 左側預覽區域
系統 SHALL 在左側顯示影片預覽視窗和播放控制。

#### Scenario: 預覽區域佈局
- **WHEN** 影片已載入
- **THEN** 左側顯示影片預覽視窗
- **AND** 預覽視窗高度最大化（佔據可用空間）
- **AND** 下方顯示播放控制（播放/暫停、進度條）
- **AND** 顯示當前時間和總時長

#### Scenario: 簡易播放進度條
- **WHEN** 影片正在播放或已載入
- **THEN** 顯示簡易播放進度條（Scrubber）
- **AND** 進度條可以拖動跳轉
- **AND** 顯示當前時間 / 總時長

### Requirement: 右側片段管理區域
系統 SHALL 在右側顯示片段列表（表格形式）、新增片段按鈕、輸出設定。

#### Scenario: 片段列表表格
- **WHEN** 存在一個或多個片段
- **THEN** 顯示表格形式的片段列表
- **AND** 表格包含列：編號、名稱、開始時間、結束時間、操作
- **AND** 每個片段顯示編輯和刪除按鈕
- **AND** 表格可以隨視窗高度延伸（可滾動）

#### Scenario: 新增片段按鈕
- **WHEN** 影片已載入
- **THEN** 顯示「新增片段」按鈕
- **AND** 點擊按鈕時，彈出片段編輯對話框
- **AND** 使用者可以輸入名稱、開始時間、結束時間後建立片段

#### Scenario: 輸出設定區域
- **WHEN** 存在片段
- **THEN** 顯示輸出設定區域
- **AND** 包含格式選擇、品質選擇、輸出目錄選擇
- **AND** 顯示批次輸出按鈕

### Requirement: 底部狀態列
系統 SHALL 在底部顯示狀態列，顯示當前應用程式狀態。

#### Scenario: 狀態列顯示
- **WHEN** 應用程式運行
- **THEN** 底部顯示狀態列
- **AND** 顯示當前狀態（就緒、載入中、匯出中等）
- **AND** 狀態列高度固定

### Requirement: 響應式設計
系統 SHALL 在不同視窗大小下都能良好顯示。

#### Scenario: 最小視窗大小
- **WHEN** 視窗大小調整
- **THEN** 確保左右分欄比例維持在合理範圍
- **AND** 設定最小視窗寬度，避免佈局過於擁擠
- **AND** 表格在空間不足時可以水平滾動（可選）
